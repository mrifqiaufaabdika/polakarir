##Tutorial

### Front-end Project setup
```
npm install
```

### Back-end Project setup
```
cd api/
composer install
```
### Run Back-end
```
cd api/
composer serve
```
### Run Front-end 
```
npm run dev

- OR- 

npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
